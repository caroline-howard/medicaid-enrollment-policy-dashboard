# Dashboard User Guide

## National Snapshot

The National Snapshot tab is designed as an executive summary page. It answers the main monitoring question:

How have national Medicaid/CHIP enrollment, applications, and eligibility determinations changed over time?

The page uses National Monitoring Summary KPI cards to show the January 2019 baseline, observed peak, latest reporting month, change since baseline, change from peak, and latest operations activity. The “What changed?” card summarizes the main enrollment movement using values from the cleaned CMS/Data.Medicaid.gov dataset.

The “Why this matters” card provides policy context. KFF and MACPAC are used only as policy context sources; CMS/Data.Medicaid.gov remains the source for dashboard metrics.

The National Snapshot includes one main annotated enrollment trend chart. Detailed applications, eligibility determinations, and application-determination balance trends are in the Eligibility Operations tab.

## Explore Next

- State Map Explorer: compare raw enrollment counts with population-adjusted enrollment and state-level changes.
- Medicaid vs CHIP Drivers: review whether enrollment changes are concentrated in Medicaid, CHIP, or both.
- Eligibility Operations: review applications, eligibility determinations, and application-determination balance as descriptive operations indicators.
- Data Quality Review: check field completeness, preliminary reporting status, and fields that require caution.

## Interpretation Limits

The dashboard supports descriptive monitoring and policy context. It does not estimate causal policy effects, measure beneficiary-level outcomes, or show claims, utilization, costs, diagnoses, access to care, or individual eligibility experiences.
