# Monitoring Questions

This project reviews official CMS/Data.Medicaid.gov Medicaid and CHIP enrollment and eligibility operations data to support descriptive monitoring across states and time. These are monitoring and exploratory questions, not causal policy evaluation questions.

## Enrollment Monitoring

- How has Medicaid/CHIP enrollment changed nationally from January 2019 through the latest available month?
- Which states show the largest enrollment increases or decreases since January 2019?
- Which states show the largest changes from the national enrollment peak or recent peak period?
- How do raw enrollment counts compare with enrollment per 1,000 residents?
- Which states may warrant follow-up review because recent enrollment changes differ from their own historical monthly pattern?

## Medicaid Vs CHIP Context

- Are changes concentrated in Medicaid enrollment, CHIP enrollment, or both?
- How does the Medicaid/CHIP split vary across selected states?
- Which states have higher Medicaid enrollment or CHIP enrollment relative to total state population?

## Eligibility Operations Monitoring

- How have applications submitted and eligibility determinations changed over time?
- Which states have higher applications per 100,000 residents?
- Which states have higher determinations per 100,000 residents?
- Which states have higher applications or determinations relative to enrollment?
- Which state-months may warrant follow-up review because applications or determinations are higher than recent state patterns?

## Data Quality Monitoring

- Which fields are complete enough for headline reporting?
- Which fields should be excluded or treated cautiously?
- Which states or months have higher missingness?
- How does preliminary/latest-month reporting affect interpretation?
- Which caveats should appear beside monitoring outputs so users do not overstate the data?

## Interpretation Boundary

This project can identify descriptive patterns, reporting caveats, and state/month review flags. It cannot explain why a change happened without additional policy, administrative, economic, demographic, and causal research design.
