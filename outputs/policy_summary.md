# Policy Summary

## What The Dashboard Helps Monitor

This dashboard helps users review official public Medicaid/CHIP reporting data across time and across states. It is designed for descriptive monitoring of enrollment, eligibility operations, state variation, and public reporting quality.

The dashboard can help an analyst identify:

- national enrollment changes from January 2019 through the latest available reporting month
- how selected states compare with the national enrollment trend
- how two selected states compare on enrollment, population-adjusted measures, applications, and determinations
- how Medicaid and CHIP enrollment contribute to state program mix
- how eligibility operations measures vary across states and months
- how fiscal-year expenditure context differs from monthly enrollment context

## How To Interpret The Main Views

### National Snapshot

The National Snapshot summarizes the broad enrollment story: January 2019 baseline, observed peak, latest reported enrollment, and change from key reference points. The indexed state-vs-national chart lets users compare trend shape even when raw state and national enrollment counts are very different.

### State Comparison Explorer

The State Comparison Explorer supports a two-state workflow. It starts with national context and then moves into direct comparison and within-state profile views. The section is useful for understanding whether a state differs because of scale, program mix, population-adjusted context, eligibility thresholds, or fiscal-year reporting context.

### Eligibility And Operations Context

Applications and determinations are descriptive operations indicators. They can show reporting activity and workflow context, but they should not be interpreted as approval rates, timeliness measures, backlog estimates, or complete performance scores.

### Fiscal Profile Context

Fiscal profile values are fiscal-year MBES/CBES expenditure reporting values. They provide context about financial scale and program categories. They should not be directly compared with one monthly enrollment value unless the time basis is clearly labeled.

## What The Dashboard Cannot Show

The dashboard cannot show beneficiary-level histories, application outcomes for specific people, county-level variation, claims, utilization, diagnoses, provider access, health outcomes, managed care plan performance, or causal effects of policy changes.

It also cannot prove that a policy caused an enrollment increase or decrease. Policy and reporting context should be interpreted alongside the descriptive trends.

## Analyst Interpretation Guidance

Use the dashboard to ask better follow-up questions:

- Is a change broad-based or concentrated in a few states?
- Is a state different because of enrollment level, population-adjusted rate, Medicaid vs CHIP mix, or operations activity?
- Is the latest month preliminary?
- Are the relevant fields complete enough for interpretation?
- Does fiscal-year expenditure context tell a different story from monthly enrollment context?

The strongest use of this project is source-aware monitoring, not causal evaluation.
